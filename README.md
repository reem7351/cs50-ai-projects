# CS50 Artificial Intelligence Projects

These are projects I completed while taking **CS50's Introduction to Artificial Intelligence with Python** from Harvard.

The course focused on core artificial intelligence concepts such as search algorithms, logical inference, machine learning, neural networks, and natural language processing.

## Projects Included

• Degrees — Graph search algorithm that finds the shortest connection between actors.

• Tic Tac Toe — AI opponent built using the **Minimax algorithm**.

• Knights — Logical inference system that solves Knights and Knaves puzzles using knowledge representation.

• PageRank — Probabilistic ranking algorithm used to evaluate page importance.

• Minesweeper — AI reasoning agent that plays Minesweeper using logical deductions.

• Crossword — Constraint satisfaction problem solver for generating crossword puzzles.

• Shopping — Machine learning classifier predicting online shopping revenue.

• Traffic — Neural network for traffic sign recognition.

• Parser — Natural language sentence parser.

• Attention — Neural network model using attention mechanisms for language tasks.

## Technologies Used

- Python
- Machine Learning
- Neural Networks
- Search Algorithms
- Natural Language Processing
